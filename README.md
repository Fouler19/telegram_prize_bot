# Telegram Prize Bot

Проект Telegram-бота с использованием библиотеки schedule и SQLite.

## Используемые технологии
- Python
- pyTelegramBotAPI
- schedule
- sqlite3
- opencv-python

## Запуск
1. Установить зависимости:
   pip install -r requirements.txt
2. Добавить токен в config.py
3. Запустить bot.py
